Bug Report: Botón "Pay Now" no se activa en formulario checkout

Descripción
El botón "Pay Now" no se activa aunque todos los campos obligatorios estén llenos correctamente y no se presenten errores de validación.

Pasos para reproducir
1. Navegar a la pagina https://demo.testim.io/.
2. Ingresar datos de partida, regreso, adultos y niños.
3. Aplicar filtro de precios y seleccionar destino.
4. Llenar todos los campos personales con datos válidos.
5. Cargar archivo y aplicar código promocional.
6. Observar que el botón "Pay Now" permanece desactivado.

Resultado esperado
El botón debería activarse al completar todos los campos válidos.

#Resultado actual
El botón permanece desactivado, impidiendo finalizar el checkout.

Evidencia
- Captura de pantalla del formulario lleno.
- Video con el flujo de prueba
